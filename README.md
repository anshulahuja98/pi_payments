# pi_payments
A Raspberry Pi 3 based payment module based on RFID.

### Technologies used:
	1.Python
	2.Raspberry Pi
	3.RFID
	4.MySQL
	5.Espeak library (Text to Speech)

### Components Required:
	1.Raspberry Pi
	2.16*2 Alphanumeric Display
	3.4*4 Numberpad
	4.EM-18 (RFID Reader)
	5.Speaker
	6.
A simple card based payment system
